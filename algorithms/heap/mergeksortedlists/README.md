# Merge K Sorted Lists

Given k linked lists, each sorted in ascending order, in a list lists, write a function to merge the input lists into
one sorted linked list.

## Examples

```text
lists = [[3,4,6],[2,3,5],[-1,6]]
3 -> 4 -> 6
2 -> 3 -> 5
-1 -> 6

Output
[-1,2,3,3,4,5,6,6]

-1 -> 2 -> 3 -> 3 -> 4 -> 5 -> 6 -> 6
```
